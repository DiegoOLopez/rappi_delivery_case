# Directorio data/

Este directorio está destinado a almacenar los archivos de datos (en bruto o procesados) que alimentan los análisis y notebooks.

> **Nota Importante:** Por motivos de peso, los archivos `.csv` y `.xlsx` de esta carpeta han sido excluidos del repositorio a través de `.gitignore`.
### Instrucciones para Desarrollo Local
Si acabas de clonar este repositorio, para poder ejecutar los notebooks sin errores, los archivos deberán descargarse y colocarse manualmente aquí con los siguientes nombres:
- `rappi_delivery_case_data - RAW_DATA.csv` (se usara para notebookks)
- `rappi_delivery_case_data.xlsx` (para referencia)
